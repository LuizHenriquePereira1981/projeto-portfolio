Projeto de Vídeos

Projeto construído para treinar a inserção de vídeos em nosso site.

Saiba mais...

Este projeto foi desenvolvido utilizando HTML5 para estruturar o conteúdo e CSS3 para estilização. O objetivo principal foi praticar a incorporação de vídeos em uma página web, garantindo que eles sejam responsivos e compatíveis com diferentes dispositivos. O projeto inclui um layout simples, onde o vídeo é centralizado e acompanhado por uma breve descrição. A experiência de usuário foi otimizada para facilitar a navegação e a visualização do conteúdo multimídia.
